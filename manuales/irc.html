IRC

Protocolo de comunicacion en tiempo real basado en texto, permite mantener conversaciones entre dos o mas personas.

Conceptos previos

IRC (Internet Relay Chat) es un protocolo de comunicacion en tiempo real, nos brinda una sala de chat, en la cual no es necesario registrarnos ni tener una relacion de contacto previo, solo basta con ingresar a la sala de chat.


Primeros Pasos

Para poder usar este protocolo, necesitaomos un servidor y conectarnos a la sala con clientes, existen una lista de servidores IRC, el mas usado Freenode, pero existen muchos mas (IRC-Hispano, etc).

El servidor provee salas (llamado comunmente canal) identificados con un nombre del tipo #canal. Para acceder a un canal, necesitamos un cliente (web o de escritorio) donde detallamos el servidor y el canal al cual queremos conectarnos (y obviamente un nickname para identificarnos).


Comandos Usados

El protocolo IRC nos permite usar comandos para diversos motivos, a continuacion ponemos los mas usados con una breve descripcion:

/kick username expulsa al usuario username del canal.

/whois username muestra la informacion del usuario username.
